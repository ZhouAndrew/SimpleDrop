# 📦 SimpleDrop

> 🚀 一款简洁高效的跨平台局域网文件传输工具，支持 Android / iOS / Windows / Linux。

---

## ✨ 项目简介

**SimpleDrop** 是一个使用 Flutter 构建的跨平台文件传输应用，支持在局域网内通过一个窗口将文件从一个设备发送到另一个设备，无需登录、注册、蓝牙或中转服务器。简单拖放，即可传送！

---

## 🎯 项目特性

- ✅ 跨平台支持：Android、iOS、Windows、Linux
- 🔍 局域网自动发现设备（基于 mDNS）
- 📤 文件拖放或浏览选择发送
- 📥 目标设备确认接收并保存
- ⚡ 即时显示传输进度
- 🔐 无需联网或云端，安全高效
